Facebook Android App Object Data SVG Choke Test

If you view this in the Facebook Android In-App browser, the browser will open for a moment then close. Browsers that work correctly will stay open and a house will be shown below.

This appears to happen because the browser crashes when viewing SVG images encoded as a data-uri into the data attribute of an object tag. This is perfectly valid HTML. Using encoded data-uri svg images is a useful technique for implementing responsive images on websites, such as Minnesota Public Radio News.
