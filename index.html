  • Biography

    I build websites and interactive experiences, from visual design to back-end programming. I'm most experienced with web publishing (blogs, news sites, etc), but have also interest and experience in data visualization and mapping.

    I am fluent in HTML, CSS, and javascript, as well as CSS-preprocessors like LESS. I prefer to build my javascript around with require.js, some jQuery if needed, and maybe even ember., and know a lot about optomizing websites for speedy client side performance. On the server, I like using python and Django, but often get stuck using php and WordPress.
